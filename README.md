# Voice Assisted Text Summarizer

A web-based NLP project that converts voice input into text, summarizes content, and performs sentiment analysis.

## Features
- Speech to text transcription
- Text summarization
- Emotion & sentiment detection
- Interactive web interface

## Tech Stack
- Python
- Flask
- NLP (Transformers)
- HTML, CSS, JavaScript

## Setup

pip install -r requirements.txt  
python app.py
